# MentorLink QA Notes

## 핵심 시나리오

### 인증

- 이메일 인증 코드 발송
- 이메일 인증 검증
- 회원가입
- 로그인
- 로그아웃
- 비밀번호 재설정

### 멘티 흐름

- 멘토 목록 조회
- 멘토 상세 조회
- 예약 가능 슬롯 조회
- 멘토링 신청
- 보낸 신청 목록 확인
- 승인 후 예정 세션 확인
- 완료 세션에서만 후기 작성

### 멘토 흐름

- 멘토 프로필 생성/수정
- 받은 신청 목록 확인
- 승인/거절
- 세션 완료 처리
- 멘토 상세에서 신청 버튼 비노출
- 신청 페이지 직접 접근 차단

### 전환 정책

- `PENDING`, `APPROVED` 신청이 있으면 멘토 전향 차단
- 예정 세션이 있으면 멘토 전향 차단
- 정리 후 멘토 전향 가능

## 확인 포인트

- 회원가입 시 `users` 반영
- 신청 시 `applications` 반영
- 승인 시 `sessions` 반영
- 후기 작성 시 `reviews` 반영
- 비밀번호 재설정 후 기존 세션 무효화

## 회귀 체크

- 프론트 `npm run lint`
- 프론트 `npm run build`
- 백엔드 `./gradlew test`
- Docker Compose 기동
- Nginx 진입 확인
